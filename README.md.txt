# SAP Budget Report Automation

## Overview

This project was developed during my internship at NLC India Limited.

The application automates SAP Project System (PS) budget reports by processing exported DAT files and generating formatted Excel reports.

## Key Features

- DAT File Processing
- Data Cleaning
- WBS Classification
- Budget Analysis
- Excel Report Generation
- Automated Formatting

## Technologies Used

- Python
- Pandas
- OpenPyXL
- XlsxWriter
- PySide6
